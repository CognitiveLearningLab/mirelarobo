Test IQ – 3 tests d'essai

Trois tests indépendants de 30 questions. Chaque test évalue cinq domaines : raisonnement fluide, numérique, verbal, visuo-spatial, mémoire de travail et attention.

Limite professionnelle : ce test n'est pas un test clinique officiel, ne remplace pas une évaluation psychométrique standardisée et ne doit pas être utilisé pour une décision médicale, professionnelle ou de sélection. Il donne un profil indicatif de performance.
